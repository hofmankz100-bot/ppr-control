# ППР Контроль PWA

Локальный адрес прототипа:

```text
http://localhost:8080/index.html
```

Что есть в прототипе:

- список оборудования;
- узлы оборудования;
- график по дням месяца;
- чек-листы ТО, ТО-2, ТО-3;
- комментарии и отметка устранения;
- красная мигающая сигнализация по оборудованию при открытом замечании;
- хранение данных в браузере через localStorage;
- manifest и service worker для PWA.

Для установки на iPhone или Android приложение нужно разместить на HTTPS-адресе.
После этого:

- iPhone: Safari -> Поделиться -> На экран Домой;
- Android: Chrome -> Установить приложение.
