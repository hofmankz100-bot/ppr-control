POSTGRESQL НА RENDER ДЛЯ v105

1. Зайди в Render Dashboard.
2. Создай PostgreSQL Database.
3. Открой свой Web Service ppr-control.
4. Settings -> Environment.
5. Добавь переменную:
   DATABASE_URL = Internal Database URL из PostgreSQL Render.
6. Сохрани и нажми Manual Deploy -> Clear build cache & deploy.

Проверка:
Открой:
/api/health

Если всё правильно, будет:
"storage":"postgresql"

Если DATABASE_URL не указан, проект продолжит работать через db.json.
