v100-stable

Добавлено без изменения рабочей логики:
1. Экспорт данных текущего месяца кнопкой «Экспорт месяца» для редактора и директора.
2. Перед экспортом сервер делает ручной backup db.json.
3. Добавлены маршруты:
   - GET /api/export/month?month=YYYY-MM
   - GET /api/export/all
   - POST /api/backup/manual
4. Backup manual хранит больше копий, чтобы месячные выгрузки не удалялись слишком быстро.
5. Обновлены версии кэша на v100.

Для флешки: нажать «Экспорт месяца», скачать ppr_export_YYYY-MM.json и скопировать файл на USB/флешку.
