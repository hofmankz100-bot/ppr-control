# ППР Контроль — запуск через интернет

Цель: чтобы QR открывался с любого интернета — заводской Wi‑Fi, мобильная сеть, дома.

## Вариант за ~$7/мес

- Render Starter — платный веб‑сервер, примерно $7/мес.
- Neon PostgreSQL — база данных, можно оставить бесплатный тариф, пока хватает лимитов.

Так сайт не должен засыпать, а все телефоны будут заходить по одному интернет‑адресу вида:

```text
https://ppr-control.onrender.com
```

## Что указать на Render

Build Command:

```text
npm install
```

Start Command:

```text
node server.js
```

Environment Variables:

```text
NODE_ENV=production
DATA_DIR=/tmp/ppr-data
DATABASE_URL=строка подключения Neon PostgreSQL
REQUIRE_POSTGRES=true
ADMIN_BOOTSTRAP_PASSWORD=сложный временный пароль админа
```

Секреты нельзя выкладывать в GitHub.

## После запуска

1. Открыть интернет‑адрес Render.
2. Проверить вход.
3. Создать тестовую запись.
4. Обновить страницу и убедиться, что запись сохранилась.
5. В приложении нажать “QR всех узлов” и распечатать новые QR.

Важно: старые QR с `10.0.0.125` после перехода в интернет больше не использовать.
