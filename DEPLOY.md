# ППР Контроль: запуск в интернете

Этот вариант работает как общий сервер: все телефоны заходят на один сайт, заявки и регистрации хранятся в общей базе `data/db.json`.

## Локальный запуск

```powershell
node server.js
```

Открыть:

```text
http://localhost:8080/?v=32
```

## Размещение на Render

1. Создайте аккаунт на `render.com`.
2. Создайте новый `Web Service`.
3. Загрузите этот проект в GitHub или подключите репозиторий.
4. Render сам прочитает `render.yaml`.
5. После запуска Render даст ссылку вида:

```text
https://ppr-control.onrender.com
```

По этой ссылке можно заходить с любого телефона и любого интернета.

## Важно

Для постоянной работы нужна платформа с постоянным диском. В `render.yaml` указан диск `/var/data`, туда сохраняется база.
